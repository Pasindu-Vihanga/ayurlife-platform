# 🌿 AyurLife – Integrated Ayurvedic & Herbal Medicine Platform

## 📖 Introduction
AyurLife is a web-based platform developed to bring Sri Lanka’s traditional Ayurvedic healthcare system into a modern digital environment.  
The main goal of this project is to connect patients, doctors, herbal suppliers, and medicine producers through a single, centralized system.

Currently, many activities in the Ayurvedic sector are handled manually, which leads to inefficiencies and lack of proper coordination.  
AyurLife is designed to address these issues by providing a more organized and accessible solution.

---

## 🎯 Objectives

- To digitalize Ayurvedic healthcare services  
- To make it easier for patients to find doctors and book appointments  
- To maintain patient medical records securely  
- To support herbal suppliers and medicine producers  
- To preserve traditional Ayurvedic knowledge in digital form  

---

## ❗ Problem Description
At present, the Ayurvedic healthcare system faces several challenges:

- Difficulty in verifying qualified doctors  
- Lack of a centralized system for maintaining patient records  
- Challenges in sourcing rare herbal ingredients  
- Inefficient management of treatments and therapy sessions  

AyurLife is developed as a solution to overcome these limitations.

---

## ⚙️ Main Features

### 👤 Patient & Treatment Management
- Manage health profiles (Prakruthi details)  
- Book appointments with Ayurvedic doctors  
- Track symptoms and monitor progress  

### 👨‍⚕️ Doctor Module
- Create Ayurvedic e-prescriptions (Digital Veda-Potha)  
- Manage patient records  
- Handle patient queues efficiently  

### 🌱 Herbal Supplier System
- Farmers can list available herbs with details  
- Enable bulk ordering (B2B system)  
- Track delivery status of herbal supplies  

### 🧪 Medicine Production & Formulation
- Manage traditional formulas such as Aristas and oils  
- Track production stages and batches  
- Maintain quality logs for consistency  

### 💆 Wellness & Therapy Management
- Schedule therapy sessions (massage, steam bath, etc.)  
- Allocate therapists and resources  
- Send post-treatment care instructions  

### 📚 Knowledge Base & Administration
- Verify doctor credentials  
- Publish Ayurvedic articles and information  
- Manage community discussions and prevent misinformation

## 🛠️ Technologies Used

- **Frontend:** React (Vite)  
- **Backend:** Node.js with Express  
- **Database:** MongoDB  
- **Deployment:** Netlify (Frontend), Railway (Backend)

## 🚀 How to Run the Project

### 1️⃣ Clone the Repository
git clone https://github.com/Pasindu-Vihanga/ayurlife-platform.git

### 2️⃣ Backend Setup
```bash
cd server
npm install
npm start
```

### 3️⃣ Frontend Setup
```bash
cd client
npm install
npm run dev
```
